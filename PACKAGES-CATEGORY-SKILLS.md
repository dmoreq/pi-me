## 🎓 Developer Guide

### Building a Successful Skill

1. **Identify Specialized Need**
   - pi-docparser: Document parsing (5.3K)
   - pi-memory-md: Persistent memory (2.8K)
   - Specific beats generic

2. **Quality Over Breadth**
   - feniix: 8 skills, 2K avg
   - vs generic service wrappers: 100-500

3. **Documentation Critical**
   - Skills require specific knowledge
   - Examples matter more than for tools

4. **Integration Depth**
   - Simple wrapper: 100-500
   - Deep integration: 2-5K

- [ ] Error handling
