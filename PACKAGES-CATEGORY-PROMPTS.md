## 🎓 Developer Guide

### Building a Successful Prompt

1. **Specific Beats Generic**
   - glimpseui (6,162) - Specific UI pattern
   - vs generic enhancer (100-400)
   - **10-50x difference**

2. **Infrastructure Value**
   - Templates (4,591) enable other work
   - Scheduling (4,642) automation
   - Enable > direct use

3. **Cross-Platform Appeal**
   - glimpseui works everywhere
   - Broad compatibility = adoption

4. **Clarity in Purpose**
   - Clear name: pi-schedule-prompt
   - Clear intent: Cron-like automation
   - Avoid ambiguity

### Publishing Checklist
- [ ] Clear purpose statement
- [ ] Specific use case
- [ ] Example invocations
- [ ] Configuration options
- [ ] Behavior documentation
- [ ] Limitations listed
- [ ] Performance implications
- [ ] Integration examples
